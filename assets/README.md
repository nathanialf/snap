# assets/

This directory is **populated at build time** by splat from your local
ROM. Its contents (other than this README and `.gitkeep`) are gitignored
and **must never be committed**.

Extracted assets here include audio banks, sequences, textures, models,
and level data. They are the property of the original rightsholders;
the build only touches them to round-trip them back into a matching ROM.
If you see anything other than this README and `.gitkeep` tracked in this
directory, open an issue tagged `legal` immediately.
